fx-CG add-ins
=============

An fx-CG add-in is a native application that can run on fx-CG and
derivatives. It can be identified using the ``G3A`` extension, and uses
the CASIO container with subtype ``USBPower\x2C\0\x01\0\x01\0``; see
:ref:`file-format-standardheader` for more information.
